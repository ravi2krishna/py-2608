    -> Python 

        -> Python is a high-level, general-purpose programming language. 
        Its design philosophy emphasizes code readability with the 
        use of significant indentation. 
        Python is dynamically type-checked and garbage-collected.


        -> Python is a high-level programming language

            -> High Level Language 

                -> High Level Language means "Human Understandable" language (English)

                -> Python Code Looks Like English

                    num2 = 2.5

                    sum = num1 + num2

                -> High Level Language means "Machine Understandable" language
